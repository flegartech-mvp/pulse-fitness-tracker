## LinkedIn
*(2-3 short paragraphs, professional but not dry)*

I built a full-featured fitness tracker called Pulse — and the entire thing runs in your browser with zero backend.

No account required. No data leaves your device. You get a 70-exercise library, a live workout session timer with automatic PR detection, body weight logging with charts, a consistency heatmap, and a goal system — all stored locally via JSON export. Dark mode default. Mobile-first. Built on React 19, TypeScript, Vite, and Tailwind CSS 4, with Playwright end-to-end tests.

This was a technical challenge: build something that feels like a polished product, not a portfolio piece, without touching a server. I think we nailed it. Source code is open — link in comments.
