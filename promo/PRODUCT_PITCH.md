# PRODUCT PITCH — Pulse

## One-Sentence Pitch

Pulse is a local-first fitness tracker that lets you build workouts, run live sessions with a built-in timer, log body weight, track personal records, and chase goals — entirely in your browser, with no account, no backend, and no data ever leaving your device.

---

## Product Description

Pulse is what a fitness app looks like when privacy is the default, not a checkbox. Built with React 19, TypeScript, Vite, and Tailwind CSS 4, it runs entirely in the browser and stores everything in local storage — meaning no sign-up, no subscription, and no server ever sees your data. Open it, start tracking, and close it. Everything is still there next time.

The feature set is deeper than most dedicated apps. You get a 70-exercise library with search and filters, a full workout builder and scheduler, a live session timer with PR detection that fires mid-workout, a body weight log with charts, a consistency heatmap, and a goal system that covers weight, strength, cardio, and consistency targets. Units toggle between kg/lb and cm/in. Dark mode is on by default. The entire app is tested with Playwright and ships mobile-first — it works as well on a phone as it does on a desktop.

---

## Best 5 Features

- **Live workout sessions** — Built-in session timer, set-by-set tracking, and automatic PR detection mid-session
- **70-exercise library** — Searchable, filterable by muscle group and equipment; per-exercise stats and history
- **Progress tracking with charts** — Body weight log, strength over time, consistency heatmap, personal records vault
- **Goal system** — Set and track body weight, strength, consistency, and cardio goals with progress indicators
- **100% local-first, no account required** — JSON data export included; nothing ever hits a server

---

## 30-Second Demo Flow

1. Open Pulse — land on the dashboard showing today's workout card, weekly stats, current streak, and a coach tip
2. Navigate to the Workout Tracker and click "Build Workout" — add 3 exercises from the library in under 15 seconds
3. Hit "Start Session" — the timer starts, log a set, and watch PR detection trigger on a new max
4. Check the Progress tab — body weight chart and consistency heatmap visible at a glance
5. Open Goals — show one active goal with its progress bar and current status
6. Open Settings — toggle from dark to light mode, switch units from kg to lb

---

## Target Audience

- Gym-goers and home athletes who are tired of apps that require an account to start tracking
- Privacy-conscious users who don't want workout data in the cloud
- Developers looking for a clean React 19 / Tailwind CSS 4 real-world project reference
- Fitness enthusiasts who want a single app for workouts, body weight, PRs, and goals — not four separate ones
